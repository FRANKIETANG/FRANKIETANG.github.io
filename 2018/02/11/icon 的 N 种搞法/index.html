背景流搞法

就是利用背景图来做 icon，优点嘛，就是不受 div 影响，不会拉伸啥的。利用 background-position 来移位置。

其实雪碧图不一定要用 Photoshop 或者 GIMP 来切，通常上会有 CSS Sprites Generator 或者用 CSS Sprites cli 。基本上 Google 一大堆教程，只要不是傻子很快就能学会的。
