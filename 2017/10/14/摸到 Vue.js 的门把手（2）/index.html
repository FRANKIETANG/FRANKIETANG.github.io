多次 import 同一个文件

Node对引入的模块都会进行缓存（缓存的是编译和执行后的对象），减少二次引入开销；
在Node的加载机制中，缓存的优先级是最高的；
这一点同时适用于不同的模块加载机制，无论ES2015的import还是CommonJS的require；
